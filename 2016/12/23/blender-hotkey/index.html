Blender热键

2016-12-23 评论 Blender学习笔记

blender 51个常用操作用的快捷键整理。

选择篇

基本 left
全选/空选 a
框选 b + left
框选（反） b + mid
涂选 c+left
涂选（反） c + mid
涂选笔触 c + scroll
自由框选 ctrl + right
自由框选（反） ctrl+shift + right
环选 alt + left
阶选 ctrl + alt + left
切换选择目标 ctrl + tab
扩张/缩小选择 ctrl + plus/minus
选择连续 ctrl + L

导航篇

旋转 mid
平移 shift + mid
缩放 ctrl + mid/scroll
相机视角 0
正交/投影 5
前/右/顶视角 1/3/7
后/左/底视角 ctrl + 1/3/7
选区视角 .
场景视角 home

视图篇

准星定位 left
准星重置 shift + c
仅当前物体 /
线框/实体 z
材质/特效 alt + z
渲染/特效 shift + z

建模篇

环切 ctrl + r
环切刀数 scroll/page updown
环切均匀 right
切割 K
切割中点吸附 ctrl
切割轴吸附 shift
切割穿透 z
内插面 i
挤出 e
斜角 ctrl + b
移动/缩放面 G + G
割裂 v
割裂并填充 alt + v
填充/连接 f

对象篇

添加物体 shift + a
移动 g 数字键
旋转 r
缩放 s
命令搜索 空格
物体锚点 ctrl + alt + shift + C
吸附 shift + s
复制 shift + d
复制并连接 alt + d
分离 p
合并 ctrl + j

动画篇

播放 alt + a
回放 shift + alt + a
逐帧 arrow
跳帧/头尾 shift + arrow
